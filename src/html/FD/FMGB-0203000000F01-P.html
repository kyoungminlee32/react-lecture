상품특징

우대서비스

  • 환율 스프레드 60% 우대 : 이 예금에 원화대가로 외화를 적립하거나 인출하는 경우
  • 현찰수수료 면제 : 1회전주기 이상 경과 후 해지하는 경우

목표환율 도달 시 자동해지

  • 예금 신규 가입 시 고객이 원하는 목표환율을 지정(매 회전주기 종료일 1영업일 전까지 변경 가능)
  • 매 회전주기 종료일 또는 최종 만기일에 ‘당행이 고시하는 2회차 환율의 전신환매입률 ≥ 목표환율’인 경우 자동해지되어 사전에 지정한 원화계좌로 입금
  • 목표환율 미지정 시, 회전주기 종료일에 환율에 관계없이 무조건 회전
  • 해당 상품은 환율변동으로 인하여 이익 또는 손실이 발생할 수 있습니다.
  • 영업점에서 외화현찰로 찾을 경우 현찰수수료가 발생합니다.
