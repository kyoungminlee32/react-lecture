ISA 계좌이전

원금손실
가능상품

계좌이전 절차 안내

  • 다른 금융기관에 ISA 계좌 또는 NH농협은행 신탁형ISA 계좌를 보유하고 계신 고객은 ISA 계약의 해지 없이 계좌를 이전해 NH농협은행의 일임형ISA에 가입하실 수 있습니다. 계좌이전제도에 따라 비과세·손익통산 등의 세제혜택이 유지되며, 기존 계약 체결일을 기준으로 하여 가입기간을 인정받을 수 있습니다.
  • 이전하고자 하는 금융기관에 방문하여 이전신청하고(일임형ISA의 경우 온라인 신청 가능), 기존 금융기관에서 전화 또는 방문을 통해 계좌이전의사를 확인하셔야 계좌이전절차가 진행됩니다.
  • 이전이 제한되는 경우
    • 압류, 가압류, 질권 설정 등의 계좌인 경우
    • 계좌이전의사 확인이 불가능한 경우
    • 리밸런싱 기간일 경우
    • 운용지시(매수/매도) 처리 진행 중인 경우
  1. 1단계
    고객
    고객이 농협은행 일임형ISA 가입을 위해 계좌이전 신청서를 작성하고 임시 신규 계좌 개설
  2. 2단계
    NH농협은행
    NH농협은행은 신청 고객의 ISA계좌가 있는 기존 금융기관에 계좌이전을 요청
  3. 3단계
    기존 금융기관
    • (전화 신청 고객) 기존 금융기관은 유선을 통해 신청고객에게 계좌이전 의사 확인
    • (방문 신청 고객) 고객이 직접 기존 금융기관 지점을 방문하여 확인
  4. 4단계
    기존 금융기관
    기존 금융기관은 NH농협은행에 이전 예정/취소 통보서 발송
  5. 5단계
    NH농협은행
    NH농협은행은 타 금융기관에 접수 통보서 발송
  6. 6단계
    기존 금융기관
    기존 금융기관은 고객자산을 환매하여 NH농협은행 계좌에 송금
  7. 7단계
    NH농협은행
    선택한 모델포트폴리오에 따라 해당 자산을 신규 매수처리
