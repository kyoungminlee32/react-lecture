실버바 가격 계산 및 수수료율

고객이 살 때 금액

1g당 기준 가격 X 중량 X (1 + 매입수수료율 19%) X (1 + 부가가치세 10%)

고객이 팔 때 금액

1g당 기준 가격 X 중량 X (1 - 매입수수료율 7%)

1g당 기준가격

국제 은 가격을 원달러 환율과 1g 당 원화 가격으로 환산하여 은행이 고시하는 가격

판매수수료

은 실물 구입 비용, 실버바 제조 비용, 보안 물류 비용, 판매업체수수료, 은행수수료 등이 포함

살 때 수수료율

㈜ 한국금거래소쓰리엠
(1g당 기준가격X중량X(1+판매수수료율 19%)X(1+부가가치세 10%))

팔 때 수수료율

1g당 기준가격 X 중량 X (1 - 매입수수료율 7%), 파실 때는 부가세 없음
